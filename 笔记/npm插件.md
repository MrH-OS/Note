## i5ting_toc

i5ting_toc是node环境下的实现工具，用于直接将markdown文件转化为网页，在浏览器打开 。

### 安装

使用方式非常简单,首先在全局安装 。

```
npm install i5ting_toc -g
```

命令参数详解

```
    -h, --help             output usage information
    -V, --version          output the version number
    -f, --file [filename]  default is README.md
    -o, --open             open in browser
    -v, --verbose          打印详细日志
```
