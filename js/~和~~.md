## ~

`~`可以将一个数a转为`-(a+1)`

```javascript
var a = 1
var b = 2
var c = 3
var d = 0
var e = -1
var f = -2
console.log('~a=' + ~a,  '~b=' + ~b, '~c=' + ~c, '~d=' + ~d, '~e=' + ~e, '~f=' + ~f)
//  ~a=-2 ~b=-3 ~c=-4 ~d=-1 ~e=0 ~f=1

var str = 'hello world'
if (~str.indexOf('he')) {
    console.log('存在')
}
if (~str.indexOf('0l')) {
    console.log('不存在')
}
```



## ~~

`~~`可以将带小数点的数值转为整数

```javascript
var a = 49.6
~~a  // 49
~~(-49.6) // 49
~~49.2  // 49
```
